Create `app/api/brief/chat/route.ts` with a POST handler that:

**1. Request parsing:**
- Parse JSON body: `{ message: string, conversation_id: string }`
- Validate `message` exists and is a non-empty string
- `conversation_id` defaults to `''` if not provided

**2. Call Dify via client:**
- Import and call `sendChatMessage` from `@/lib/dify/client`
- Pass `{ query: message, conversation_id }`
- Wrap in try/catch for error handling

**3. Error handling (use exact copy from UI-SPEC.md):**
- If `sendChatMessage` throws with message containing "not configured":
  ```json
  { "error": "Chat service is not configured. Please contact the administrator.", "code": "DIFY_NOT_CONFIGURED", "status": 500 }
  ```
  Return HTTP 500.

- If `sendChatMessage` throws with message containing "Dify API error 429":
  ```json
  { "error": "Too many requests. Please wait a moment before sending another message.", "code": "DIFY_RATE_LIMIT", "status": 429 }
  ```
  Return HTTP 429.

- If `sendChatMessage` throws (any other error):
  ```json
  { "error": "Unable to reach the chat service. Please try again in a moment.", "code": "DIFY_UNREACHABLE", "status": 502 }
  ```
  Return HTTP 502.

**4. SSE stream transformation:**
- Get the response body reader from Dify's Response
- Create a `ReadableStream` that:
  - Uses a `TextDecoder` to decode incoming chunks
  - Buffers partial data, splits on `\n` boundaries
  - For each line starting with `data:`, strips the prefix and parses JSON
  - Handles both `message` AND `agent_message` events (same handler -- both have `answer` field):
    ```
    data: {"answer":"chunk text","conversation_id":"uuid","message_id":"uuid"}\n\n
    ```
  - Handles `message_end` event:
    ```
    data: {"event":"done","conversation_id":"uuid"}\n\n
    ```
  - Handles `error` event:
    ```
    data: {"event":"error","message":"Something went wrong with the chat service. Please try again."}\n\n
    ```
  - Silently skips malformed JSON (try/catch around JSON.parse, continue on error)
  - Calls `controller.close()` when reader is done

**5. Response headers:**
```typescript
return new Response(stream, {
  headers: {
    'Content-Type': 'text/event-stream',
    'Cache-Control': 'no-cache',
    'Connection': 'keep-alive',
  },
});
```

**Critical anti-patterns to avoid:**
- Do NOT add `export const runtime = "edge"` -- use Node.js runtime (default)
- Do NOT use Vercel AI SDK `streamText` -- Dify SSE format is incompatible
- Do NOT store conversation state server-side -- just pass through conversation_id
- Do NOT assume one TCP chunk = one SSE event -- always buffer and split on `\n`

**Full reference implementation** (adapt from research, with error handling from UI-SPEC):

```typescript
import { sendChatMessage } from '@/lib/dify/client';

export async function POST(req: Request) {
  try {
    const body = await req.json();
    const message = body.message;
    const conversation_id = body.conversation_id || '';

    if (!message || typeof message !== 'string' || message.trim() === '') {
      return new Response(
        JSON.stringify({ error: 'Message is required', code: 'INVALID_REQUEST', status: 400 }),
        { status: 400, headers: { 'Content-Type': 'application/json' } }
      );
    }

    let difyResponse: Response;
    try {
      difyResponse = await sendChatMessage({ query: message.trim(), conversation_id });
    } catch (err) {
      const errMsg = err instanceof Error ? err.message : '';
      if (errMsg.includes('not configured')) {
        return new Response(
          JSON.stringify({ error: 'Chat service is not configured. Please contact the administrator.', code: 'DIFY_NOT_CONFIGURED', status: 500 }),
          { status: 500, headers: { 'Content-Type': 'application/json' } }
        );
      }
      if (errMsg.includes('429')) {
        return new Response(
          JSON.stringify({ error: 'Too many requests. Please wait a moment before sending another message.', code: 'DIFY_RATE_LIMIT', status: 429 }),
          { status: 429, headers: { 'Content-Type': 'application/json' } }
        );
      }
      return new Response(
        JSON.stringify({ error: 'Unable to reach the chat service. Please try again in a moment.', code: 'DIFY_UNREACHABLE', status: 502 }),
        { status: 502, headers: { 'Content-Type': 'application/json' } }
      );
    }

    const reader = difyResponse.body!.getReader();
    const decoder = new TextDecoder();
    const encoder = new TextEncoder();

    const stream = new ReadableStream({
      async start(controller) {
        let buffer = '';
        try {
          while (true) {
            const { done, value } = await reader.read();
            if (done) break;

            buffer += decoder.decode(value, { stream: true });
            const lines = buffer.split('\n');
            buffer = lines.pop() || '';

            for (const line of lines) {
              if (line.startsWith('data:')) {
                const data = line.slice(5).trim();
                if (!data) continue;
                try {
                  const parsed = JSON.parse(data);
                  if (parsed.event === 'message' || parsed.event === 'agent_message') {
                    controller.enqueue(encoder.encode(
                      `data: ${JSON.stringify({ answer: parsed.answer, conversation_id: parsed.conversation_id, message_id: parsed.message_id })}\n\n`
                    ));
                  } else if (parsed.event === 'message_end') {
                    controller.enqueue(encoder.encode(
                      `data: ${JSON.stringify({ event: 'done', conversation_id: parsed.conversation_id })}\n\n`
                    ));
                  } else if (parsed.event === 'error') {
                    controller.enqueue(encoder.encode(
                      `data: ${JSON.stringify({ event: 'error', message: 'Something went wrong with the chat service. Please try again.' })}\n\n`
                    ));
                  }
                } catch { /* skip malformed JSON */ }
              }
            }
          }
        } finally {
          controller.close();
        }
      }
    });

    return new Response(stream, {
      headers: {
        'Content-Type': 'text/event-stream',
        'Cache-Control': 'no-cache',
        'Connection': 'keep-alive',
      },
    });
  } catch (error) {
    return new Response(
      JSON.stringify({ error: 'Something went wrong with the chat service. Please try again.', code: 'DIFY_ERROR', status: 502 }),
      { status: 502, headers: { 'Content-Type': 'application/json' } }
    );
  }
}
```

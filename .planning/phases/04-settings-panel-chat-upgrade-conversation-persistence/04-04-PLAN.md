API Routes (from Plan 03):
```
GET  /api/agents          -> Agent[] (apiKey masked)
POST /api/agents          -> Agent (body: {label, apiKey, baseUrl?, difyUrl?, conversationMode?})
GET  /api/agents/:id      -> Agent
PATCH /api/agents/:id     -> Agent (body: partial fields, isActive toggle)
DELETE /api/agents/:id    -> {success: true}

GET  /api/test-prompts    -> TestPrompt[] (ordered by displayOrder)
POST /api/test-prompts    -> TestPrompt (body: {name, text, autoSend?, displayOrder?})
GET  /api/test-prompts/:id -> TestPrompt
PATCH /api/test-prompts/:id -> TestPrompt
DELETE /api/test-prompts/:id -> {success: true}
```

Agent shape:
```typescript
{ id, label, apiKey (masked), baseUrl, difyUrl, conversationMode, isActive, createdAt, updatedAt }
```

TestPrompt shape:
```typescript
{ id, name, text, autoSend, displayOrder, createdAt, updatedAt }
```

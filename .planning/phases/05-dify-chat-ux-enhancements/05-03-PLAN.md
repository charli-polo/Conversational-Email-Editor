- Second root (hover visible): `autohide="always"` with Copy, Reload, ExportMarkdown
  - Copy with tooltip "Copy to clipboard" using Copy icon
  - Reload with tooltip "Regenerate response" using RefreshCw icon — use `autohide="not-last"` on a SEPARATE third ActionBarPrimitive.Root so it only shows on the last message per D-01

Actually per the research, `autohide="not-last"` on ActionBar restricts the ENTIRE bar to last message. So structure should be:
- Root 1 (no autohide): feedback buttons — always visible on all assistant messages
- Root 2 (`autohide="always"`): Copy + ExportMarkdown — hover on any assistant message
- Root 3 (`autohide="not-last"`): Reload only — hover only on last assistant message

All three wrapped in a container `<div className="flex flex-wrap gap-1 mt-1">`.

```tsx
function AssistantMessageContent() {
  const message = useMessage();
  const metadata = (message as Record<string, unknown>)?.message?.metadata;
  const custom = (metadata as Record<string, unknown>)?.custom as Record<string, unknown> | undefined;
  const isOpener = custom?.isOpener === true;
  const suggestedQuestions = (custom?.suggestedQuestions as string[]) || [];
  
  // Extract reasoning parts from content
  const content = (message as Record<string, unknown>)?.message?.content;
  const reasoningParts = Array.isArray(content)
    ? content.filter((p: Record<string, unknown>) => p.type === 'reasoning').map((p: Record<string, unknown>) => ({ text: p.text as string }))
    : [];

  return (
    <>
      <div className="w-full text-sm text-foreground prose prose-sm dark:prose-invert max-w-none prose-p:my-1 prose-ul:my-1 prose-ol:my-1 prose-li:my-0.5 prose-headings:my-2">
        <MessagePrimitive.Content
          components={{
            Text: ({ text }) => <Markdown>{text}</Markdown>,
          }}
        />
      </div>
      {reasoningParts.length > 0 && <ReasoningSection reasoningParts={reasoningParts} />}
      {isOpener && suggestedQuestions.length > 0 && <OpenerSuggestions suggestions={suggestedQuestions} />}
      <AssistantActionToolbar />
    </>
  );
}
```

Then in BriefMessage:

  </div>
</MessagePrimitive.If>

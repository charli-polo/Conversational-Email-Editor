## Decisions Made

**2. [Rule 1 - Bug] Strict mode violation on rename/delete buttons**
- **Found during:** Task 1 (LIST-03/LIST-04 tests)
- **Issue:** `page.getByTitle('Rename conversation')` resolved to multiple elements because serial tests share DB state with previously seeded conversations
- **Fix:** Scoped button locators to the specific `.group` row containing the target conversation title
- **Files modified:** __tests__/e2e-phase8.spec.ts
- **Committed in:** 83afac4

**3. [Rule 1 - Bug] Edit input not found within row locator**
- **Found during:** Task 1 (LIST-03 rename test)
- **Issue:** `row.locator('input')` failed because `.group` filter with `hasText: 'Old Name'` lost match when the title span was replaced by an input (input value is not text content)
- **Fix:** Used page-level `.divide-y input` locator since only one edit input exists at a time
- **Files modified:** __tests__/e2e-phase8.spec.ts
- **Committed in:** 83afac4

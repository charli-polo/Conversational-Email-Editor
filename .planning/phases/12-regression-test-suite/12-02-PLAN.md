<!-- API endpoints -->
- GET /api/threads — list conversations
- POST /api/threads — create conversation (returns { id })
- PATCH /api/threads/{id} — rename (body: { title })
- DELETE /api/threads/{id} — delete conversation
- POST /api/threads/{id}/tags — assign tag (body: { name })
- DELETE /api/threads/{id}/tags — remove tag (body: { name })
- GET /api/agents — list agents
- POST /api/agents — create agent (body: { label, apiKey, baseUrl?, difyUrl?, conversationMode?, isActive? })
- DELETE /api/agents/{id} — delete agent
